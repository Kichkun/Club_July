Папка ESP - исходный код для ESP8266-12 
ESP/data - сайт, надо грузить в ESP при помощи ESP8266 Sketch data uploader
изображение надо пулучить с arduino и отправить по байтно при помощи следующего кода:
int xx[1] = {data};
webSocket.broadcastBIN((uint8_t *)(xx), 1);
По завершению приема 320*240 пикселей - изображение будет отображено на сайте


ImageDecoder - папка с python-скриптом для декодирования изображений с карты в png
нужно поставить PIL и запустить main.py


OV7670_SD0 - код для получения изображения с камеры и сохранения на карту, надо доработать передачу на ESP
